📖 Setup Guide — TeleRelay
① Create a Telegram Bot
1
Open Telegram and search for @BotFather, then start a chat.
2
Send /newbot and follow the prompts to name your bot.
3
Copy the Bot Token (e.g. 7123456789:AAHxxxx...) and paste it into the Bot Token field.
4
Disable Privacy Mode — send /setprivacy to @BotFather → select your bot → choose Disable. Without this the bot cannot read group messages.
② Create a Telegram Group
1
Create a new group. Add yourself, your partner, and the bot.
2
Send any message in the group (e.g. hello) so the bot registers an update.
③ Find Your Group Chat ID
1
Open this URL (replace TOKEN):
https://api.telegram.org/botTOKEN/getUpdates
2
Find "chat":{"id":-100xxxxxxxxxx} — that negative number is your Chat ID.
💡 Or add @RawDataBot to the group — it posts the Chat ID automatically.
④ Fill in the Setup Form
1
Bot Token — from @BotFather.
2
Group Chat ID — the negative number above.
3
Display Name — e.g. Husband or Wife.
4
Shift Key — 4+ numbers separated by commas (e.g. -41,-19,11,5). Encrypts your token. Write it down!
5
Click Save & Start Chat — done! 🎉
⑤ Share Access via QR Code (optional)
1
Click Save & Generate QR to create a login QR.
2
Download & store it securely (printed in wallet, or encrypted folder).
3
On another device, click Scan QR Code — logged in instantly, no typing needed.
⚠️ QR contains your token in plain text — keep it private.
